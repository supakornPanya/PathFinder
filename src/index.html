Welcome to my Graph Pathfinding Tool!
This website is designed to help you explore and solve pathfinding problems in a grid-based graph. The grid consists of rows and columns, with each cell representing a node. You can customize the grid by defining valid and invalid nodes and calculate optimal paths.

Each node is defined by its position: x represents the node's position along the x-axis, and y represents the position along the y-axis. Nodes are also color-coded for clarity:

  • Green nodes are valid and can be traversed.
  • Red nodes are invalid and cannot be traversed.

    • Tip: Click on any node in the grid to toggle its type.
